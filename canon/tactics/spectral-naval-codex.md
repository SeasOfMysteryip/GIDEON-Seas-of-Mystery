# Spectral Naval Combat Codex

This file contains the core named maneuvers Gideon should treat as locked tactical doctrine.

## Tactical Laws

- Tactics are not interchangeable spectacle. Each named maneuver carries a tactical purpose, typical users, and a spectral twist.
- Spectral twists should intensify the maneuver, not replace its tactical logic.
- Named maneuvers should preserve faction identity. Ching should not fight like Vane, and Murakami should not fight like Jeb.

## Named Maneuvers

### The Bladebreaker Maneuver
- Used by: Jeb Blackwater, Old Red, Captain Lysara in legend
- Core tactic: break enemy line and shatter ley circuit
- Spectral twist: timed fracture ignition and tuning-pulse panic flare
- Canon tag: `#SpectralTactic_Bladebreaker`

### The Driftburn Gambit
- Used by: Vaultborn Reapers, Revisionist cults
- Core tactic: sacrificial fire-ship detonation
- Spectral twist: dead ship sings before ignition
- Canon tag: `#SpectralTactic_DriftburnGambit`

### The Pulse Choke
- Used by: Ching Shih, Ghostfleet Harbormasters
- Core tactic: funnel enemy into ley choke and seal exit
- Spectral twist: false bearings and ghost-map corruption
- Canon tag: `#SpectralTactic_PulseChoke`

### The Fog Rattle
- Used by: Murakami Hideyoshi, Flame of Sidon
- Core tactic: strike during ley fog and waking confusion
- Spectral twist: fractured time-sense before awareness
- Canon tag: `#SpectralTactic_FogRattle`

### The Ghostlash Feint
- Used by: Lark Hawthorne, Revisionist saboteurs
- Core tactic: false safe zone and rear strike
- Spectral twist: phantom fleets and projection assault
- Canon tag: `#SpectralTactic_GhostlashFeint`

### The Marrowline Cut
- Used by: Vernon, Ley Corsairs
- Core tactic: target relic-bearers, not whole crews
- Spectral twist: shell-plus-chronometer camouflage
- Canon tag: `#SpectralTactic_MarrowlineCut`

### The Shatterwake Coil
- Used by: Blackbeard's ringbearers, Deep Vault hunters
- Core tactic: spiral prey and collapse pulse field
- Spectral twist: multiplied ship-image distortion and hull feedback fracture
- Canon tag: `#SpectralTactic_ShatterwakeCoil`

### The Leviathan Pulse
- Used by: The Bloody Gull from EP106 forward
- Core tactic: relic-triad precision ley rupture
- Spectral twist: can erase memory, displace time, or unwrite a moment
- Canon tag: `#SpectralTactic_LeviathanPulse`

### The False Flag Spiral
- Used by: Flame of Sidon, Mirage Fleets
- Core tactic: surrender or signal deception
- Spectral twist: counterfeit relic frequencies and projected familiar voices
- Canon tag: `#SpectralTactic_FalseFlagSpiral`

### The Crown's Reckoning
- Used by: Ching Shih
- Core tactic: end a fight through sovereign psychological collapse
- Spectral twist: enemy relics go dark and black water rises without contact
- Canon tag: `#SpectralTactic_CrownsReckoning`

## Tactical Failure Test

If a scene uses a named maneuver but strips out its tactical signature, faction identity, or lawful spectral twist, Gideon should flag it.
