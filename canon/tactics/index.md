# Tactics Canon Index

This index defines the tactical doctrine files extracted from the Factions, Fleets & Tactics module.

## Authority Rule

Use the files in this folder as the source of truth for:

- named naval maneuvers
- ship-specific Bloody Gull combat moves

Keep cinematic flourish and historical battle inspiration separate from hard tactical law.
