# Tactics Canon Index

This index defines tactical doctrine extracted and cleaned from the Factions, Fleets & Tactics source material.

## Authority Rule

Use the files in this folder as the source of truth for:

- named naval maneuvers
- ship-specific Bloody Gull combat moves
- faction fleet maneuver chains
- tactical signatures used in battle-scene review
- historical inspiration patterns kept separate from hard canon

## Files

| File | Function |
| --- | --- |
| spectral-naval-codex.md | core named naval maneuvers, costs, users, and tactical laws |
| bloody-gull-tactics.md | Jeb's ship-specific combat doctrine |
| maneuver-chains.md | multi-fleet battle chains and formation protocols |
| historical-battle-references.md | reference-only inspiration source, not hard canon |

## Failure Test

If a battle scene treats tactics as names for spectacle rather than battlefield logic with faction temperament and cost, Gideon should flag it.
