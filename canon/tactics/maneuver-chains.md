# Fleet Maneuver Chains

Named maneuver chains are faction identity expressed as battle logic.

## Core Chains

### The Coil of Dread
- Faction: Blackbeard's Fleet
- Objective: dominate, control relic, leave no survivors
- Pattern: fog split, false-vector flank, silence push, relic extraction strike

### The Phantom Spiral
- Faction: Charles Vane's Spectral Fleet
- Objective: disruption, reaction-testing, relic theft if possible
- Pattern: ghost-feint, false-destruction decoy, memory hallucination pressure

### The Boneclock Trap
- Faction: Murakami's Hunters
- Objective: target Blackbeard directly and extract relics or data
- Pattern: out-of-sync strike plus temporal minefield

### The Bladewall Protocol
- Faction: Admiral Vernon's Fleet
- Objective: impose order, break pirate factions, capture relic
- Pattern: line control, shield screen, disruptor barrage

#### Crown Crucible
- Phase-two pattern under Bladewall
- Goal: freeze advances and isolate duels

#### Judgment Column
- Final-phase pattern under Bladewall
- Goal: split, contain, and collapse enemy formation inward

### The Serpent's Reversal
- Faction: Darak al-Mhar's Fleet
- Objective: opportunistic disruption and reversal
- Pattern: misdirection, baiting, reverse-flow attack

### The Queen's Verdict
- Faction: Ching Shih
- Objective: late-arrival dominance and sovereign end-state control
- Pattern: finality through silence and command pressure

## Chain Failure Test

If a fleet chain is used without preserving faction temperament, objective, and pattern logic, Gideon should flag it.
