# Fleet Maneuver Chains

Named maneuver chains are faction identity expressed as battle logic. Use these to keep fleet scenes from becoming interchangeable cannon chaos.

## Core Chains

### The Coil of Dread

- Faction: Blackbeard's fleet.
- Objective: dominate the battle space, control relic access, and leave survivors psychologically broken if not dead.
- Pattern: fog split, false-vector flank, silence push, relic extraction strike.
- Temperament: quiet gravity. The enemy should feel surrounded before they understand by what.
- Failure sign: Blackbeard's fleet behaves like noisy raiders instead of inevitability with teeth.

### The Phantom Spiral

- Faction: Charles Vane's ghost fleet.
- Objective: disrupt, terrify, test reactions, steal relic advantage if rage leaves room.
- Pattern: ghost-feint, false-destruction decoy, memory hallucination pressure, violent reveal.
- Temperament: wrath, fire, and emotional ruin.
- Continuity note: Vane dies in EP201, so later use of this chain should be ghost-fleet consequence or inherited tactical memory unless explicitly staged otherwise.
- Failure sign: the chain feels disciplined, rational, or clean.

### The Boneclock Trap

- Faction: Murakami's hunters.
- Objective: target a power center directly and extract relics, data, or command advantage.
- Pattern: out-of-sync strike, temporal minefield, precision isolation, clean exit.
- Temperament: patience, calculation, and no wasted violence.
- Failure sign: Murakami fights from anger or spectacle rather than design.

### The Bladewall Protocol

- Faction: Admiral Vernon Harrow's imperial fleet.
- Objective: impose order, break pirate flexibility, capture or disable relic functions.
- Pattern: line control, shield screen, disruptor barrage.
- Temperament: ceremony, obedience, and dead imperial law.
- Failure sign: Vernon chases like a pirate instead of waiting where order should be.

#### Crown Crucible

- Phase-two pattern under Bladewall.
- Setup: flagship holds center, shield escort rotates to cover flanks, disruption ship arcs wide.
- Goal: freeze advances, disable relic weaponry, break pirate lines into isolated duels.
- Symbolic logic: empire as crucible; all must pass through fire and be judged.

#### Judgment Column

- Final-phase pattern under Bladewall.
- Setup: flagship splits the enemy line while escorts form a collapsing perimeter.
- Goal: cut, contain, and collapse enemy formation inward.
- Spectral behavior: vertical positioning through draglines can create battlefield compression without turning the scene into full aerial combat.
- Symbolic logic: the column does not merely destroy; it judges.

### The Serpent's Reversal

- Faction: Darak al-Mhar's fleet.
- Objective: opportunistic disruption, reverse-flow attack, profit from someone else's certainty.
- Pattern: misdirection, bait signal, false relic lock, reverse-flow strike.
- Temperament: charm, hustle, and profitable betrayal.
- Failure sign: Darak fights nobly or straightforwardly.

### The Queen's Verdict

- Faction: Ching Shih.
- Objective: late-arrival dominance and sovereign end-state control.
- Pattern: silence, pressure, relic darkening, surrender collapse.
- Temperament: finality.
- Failure sign: Ching Shih uses Blackbeard-style theater or Vane-style fury.

## Chain Failure Test

If a fleet chain is used without preserving faction temperament, objective, formation logic, and consequence, Gideon should flag it.
