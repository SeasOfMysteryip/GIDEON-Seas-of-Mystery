# Relics Canon Index

This folder contains the relic doctrine and relic-specific canon files extracted from the Relics module.

## Authority Rule

Use these files as the source of truth for:

- what relics are in-world
- what relics mean
- bearer and bloodline rules
- relic cost and limitation
- relic symbiosis
- circuit behavior
- faction pressure around relics

Apply language-normalization.md before treating any raw source wording as canon.

## Files

| File | Function |
| --- | --- |
| language-normalization.md | naming and cleanup rules for relic conversion |
| relic-mirror-doctrine.md | core philosophy, meaning, and human purpose of relics |
| relic-circuit.md | node map, flow logic, and factional desire around the full relic system |
| active/index.md | index of active Season One relics and their file map |
| dormant/index.md | index of dormant and future relics that remain outside active Season One use |
