# Abyssal Chronometer

## Core Identity

The Abyssal Chronometer is a counterfeit Phoenician temporal lure: useful, dangerous, and spiritually ungrounded. It imitates circuit behavior without true harmony.

## Function

- detects unstable fracture seams
- enables short, high-risk temporal slips
- overcharges nearby relics for temporary clarity
- acts as bait, pressure amplifier, and misdirection device

## Limitation And Cost

- amplifies panic, fixation, and fracture addiction
- emits predator-drawing signals
- causes identity bleed and temporal dislocation
- cannot be mastered cleanly by anyone

## Bearer Rule

No bloodline lock. Many can handle it. None should own it safely. It is most dangerous when mistaken for a true system relic.

## Symbiosis

- cross-flares with Compass, Spyglass, and Shell
- nearly destabilizes grid logic when forced into sync
- later fuses into the Gull as a sub-core state after Red's death pulse

## Faction Pressure

Darak weaponizes it as bait and manipulation. Murakami seeks to force system-level use out of it. Both should be treated as dangerous misreaders rather than rightful bearers.

## Episode Status

Active pressure relic through Episodes 104 to 108, then transformed through fusion into the Gull's altered relic state.

## Failure Test

Flag drift if the Chronometer is treated like a stable navigation relic, a clean circuit node, or a harmless timing device.
