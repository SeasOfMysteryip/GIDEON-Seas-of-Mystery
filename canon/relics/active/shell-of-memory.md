# Shell of Memory

## Core Identity

The Shell of Memory is a language and revelation relic. It does not invent truth. It exposes buried memory, signal, and Library-facing resonance through partial, emotionally charged transmission.

## Function

- translates ley pulses, fracture pressure, and memory-coded systems
- opens auditory communion with buried systems, dead memory, and spectral response
- unlocks sound-based relic systems
- recovers ancestral or personal memory at dangerous cost

## Limitation And Cost

- reveals only partial truth
- scars Sophia through glyph burn and pulse dependency
- blurs identity through memory corruption
- deepens emotional drift and ancestral pressure with repeated use

## Bearer Rule

Sophia is the primary bloodline user. The Shell should feel intimate and invasive, not like a neutral lore terminal.

## Symbiosis

- decodes the `Compass of Ley Currents`
- clarifies `Spyglass of Storms`
- spikes with Old Red's dice and later with the Gull itself
- harmonizes with the ship after Red's death

## Faction Pressure

Memory-seeking powers, Murakami, and Library-facing interpreters all want what the Shell can reveal, but none should access it as cleanly as Sophia.

## Episode Status

Season One active relic. It remains central to Sophia's memory burden and to Library-facing system activation.

## Failure Test

Flag drift if the Shell gives clean exposition, reveals without distortion, or acts without emotional consequence.
