# Memory of Kings

## Core Identity

Memory of Kings is a living map and authored-memory relic. It records, reveals, and tempts inscription. Reading is one thing. Writing is risk.

## Function

- maps fracture routes and ley pathways
- records past, present, and unfolding memory pressure
- projects bloodline vision and future possibility
- allows inscription that can thread fate into being

## Limitation And Cost

- tempts the bearer to become an editor of destiny
- burdens every inscription with moral consequence
- causes waking vision bleed and future-pressure
- punishes non-bloodline handling with distortion and chaos

## Bearer Rule

Lark is the primary stabilizing bearer. Elias can transmit through it. Sophia can sharpen it, but not command it.

## Symbiosis

- clarifies and is clarified by the Spyglass
- syncs with the Tuning Core and later the Gull itself
- intensifies navigation pressure around the Compass

## Faction Pressure

Murakami and any power seeking authored history or controlled routes would treat it as an ultimate weapon. Gideon should preserve that obsession.

## Episode Status

Found in Episode 105. Guides key route and battle decisions through Episodes 106 to 110. Seeds a larger fate-versus-trust dilemma for later seasons.

## Failure Test

Flag drift if the map behaves like passive reference material, rewrites fate without cost, or loses the moral burden of inscription.
