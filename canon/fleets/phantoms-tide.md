# Phantom's Tide

## Core Identity

- Type: ghost ship; apex predator echo
- Captain: Charles Vane, deceased
- Status: posthumous echo and terror vessel
- Alignment: Vane's ghost-fleet legacy

Phantom's Tide is less a ship than judgment written in spectral flame. After Vane's death, its use should read as haunting consequence, not an active living command unless the story explicitly earns that exception.

## Tactical Signature

Terror without touch.

## Traits And Behavior

- No sound and no wake.
- Green spectral flame on rigging.
- Triggers relic echoes involuntarily.
- Survivors carry glyph scars or memory voids.

## Fleet Behavior

- Attacks memory before hull.
- Makes crews remember seeing it before it arrived.

## Primary Tactics

- Ghostlash Feint
- Memory Drown
- spectral recoil drift

## Supporting Ships

- Broken Promise: false-destruction decoy.
- Mercy's Price: memory and madness pressure vessel.

## Gideon Failure Test

If Phantom's Tide feels like a normal commanded ship after EP201, Gideon should flag it.
