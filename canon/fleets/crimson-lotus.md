# Crimson Lotus

## Core Identity

- Type: spectral empress warship
- Captain: Ching Shih
- Status: fleet command vessel
- Alignment: Ching Shih's abyssal command

Crimson Lotus blooms in fire and fracture storms. It is not just a ship; it is Ching Shih's command logic made visible.

## Tactical Signature

Control by silence.

## Traits And Behavior

- Scarlet hull.
- Lotus-petal sails.
- Relic fragments chained along decks.
- Heartbeat-like war drums below deck.

## Fleet Behavior

- Commands fleets by ley pulse.
- Petals fold into defensive ley shields.
- Arrives as finality rather than pursuit.

## Primary Tactics

- Pulse Choke
- Crown's Reckoning
- Black Wake Spiral as controlled drag


## Gideon Failure Test

If Crimson Lotus behaves like a loud raider or Blackbeard-style dread ship, Gideon should flag it.
