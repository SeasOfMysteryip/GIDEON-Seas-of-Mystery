# Fleet Canon Index

This folder contains flagship and fleet doctrine files. Fleet files define how ships behave as faction expression in battle, not just what they look like.

## Authority Rule

- Character profiles govern captains.
- Fleet profiles govern ship behavior and battle identity.
- Tactics files govern named maneuvers and formation logic.
- The USS Cyclops remains a separate Season Two / Benjamin Briggs file and should not be bulk-edited through this fleet pass.
