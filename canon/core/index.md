# Core Canon Index

This folder contains the constitutional layer of Seas of Mystery.

Use these files as the source of truth for:

- what kind of series this is
- how the story should feel
- what themes must stay alive
- how the season spine is shaped
- where high-level world doctrine ends and Deep Myth begins

## Authority Rule

`canon/core/` governs the series identity.

`canon/deep-myth/` governs the hidden machinery beneath that identity.

Core files may summarize Deep Myth, but they should not duplicate its full law.

## Files

| File | Function |
| --- | --- |
| `series-constitution.md` | defines the non-negotiable identity of the series |
| `convergence-doctrine.md` | summarizes the hidden-world doctrine at a constitutional level and points to Deep Myth for system law |
| `tone-and-format.md` | defines format, sensory signature, visual law, and storytelling posture |
| `themes-and-motifs.md` | defines the major thematic engines and recurring motifs |
| `oral-tradition-and-song.md` | defines how songs, chants, and oral culture function in-world without turning all lyrics into literal canon law |
| `season-spine.md` | captures the top-level series overview and season-one spine |
| `timeline-overview.md` | captures the high-level chronology layer and continuity posture |
