# Core Canon Index

This folder contains the constitutional layer of Seas of Mystery.

Use these files as the source of truth for:

- what kind of series this is
- how the story should feel
- what themes must stay alive
- how the season spine is shaped
- how Season One's main plotline logic works
- where high-level world doctrine ends and Deep Myth begins

## Authority Rule

canon/core/ governs the series identity.

canon/deep-myth/ governs the hidden machinery beneath that identity.

Core files may summarize Deep Myth, but they should not duplicate its full law.

Older Season Two material in the core bible is superseded by season-planning/s02/outline-frame.md.
