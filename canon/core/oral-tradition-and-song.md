# Oral Tradition And Song

Songs, chants, and repeated deck-tales are part of how `Seas of Mystery` stores memory inside the world.

## Canon Rule

Song in this setting may function as:

- warning language
- crew tradition
- relic stabilization or tuning behavior
- memory preservation
- faction or ship identity

It should not be treated as random decoration.

## Old Red Rule

Old Red's voice may legitimately draw from:

- warning-song fragments
- broken shanty lines
- marrow-cycle refrains
- rhythmic omen language

That sourcing should feel inherited, not improv-comedy and not generic pirate verse.

## Bloody Gull Rule

The Bloody Gull carries oral culture.

That means crew stories, chants, and repeated lines can:

- preserve prior battles
- encode warnings
- carry morale and rebellion memory
- keep the ship feeling lived-in across time

## Boundary Rule

Not every source-lore lyric is hard canon.

What becomes canon-relevant is:

- the function of song
- recurring adopted refrains
- any line or pattern already embedded in scripts or locked canon files
