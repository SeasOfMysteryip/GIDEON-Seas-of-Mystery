# Episode 102 Index

Episode 102 is the first full episode scaffold built in the repo-native format.

Title: `The Broken Compass`

## Purpose

This folder separates:

- the clean episode script
- continuity and canon locks
- revision history
- scene-card logic adapted from the Forge workflow
- source references back to the whole-episode and by-scene documents

## Authority Rule

For episode-specific review:

1. `script.md` is the clean episode text source.
2. `continuity-notes.md` holds episode-level canon locks and establishment beats.
3. `scene-cards/` holds scene function, pulse, present canon elements, and transition logic.
4. `revision-notes.md` records late-pass and reader-note shaping.

If any episode file conflicts with locked canon in `canon/`, locked canon wins.
