## Repo-Native Rule

Scene cards should preserve:

- scene ID
- scene title or location heading
- scene pulse
- present characters
- present relics / factions / tactics where relevant
- emotional role map
- continuity hooks
- entry / exit function

Scene cards should not preserve:

- GPT protocol wrappers
- repeated command scaffolding
- prohibition boilerplate
- duplicated formatting-enforcer language
