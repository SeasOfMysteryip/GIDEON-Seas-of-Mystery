# Tales Index

This folder preserves flash fiction, deck tales, oral-history fragments, Bloody Gull legends, and early source narratives from the origin layer.

## Use Rule

Tales are as important as songs. Use them as origin-history source lore for Captain Jeb, the Bloody Gull, the crew's storytelling culture, and adapted character functions. A lot of Jeb's monologue DNA lives here.

## Adaptation Rule

Lark, Gus, Old Red, Stumpy Pete, Jeb, and the Bloody Gull crew may inherit functions, pressures, speech rhythms, or scenario logic from these tales even when the current canon version has changed names, roles, or continuity.

## Authority Rule

Treat tales as source material unless a specific beat, saying, superstition, or event has been promoted into a canon file, imported episode, continuity note, or approved scene draft.
