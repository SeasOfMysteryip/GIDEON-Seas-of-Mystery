# Tales Index

This folder is for flash-fiction, oral-history, survivor fragments, deck myths, and short lore narratives from the source layer.

## Use

These tales appear to feed:

- character texture
- Bloody Gull storytelling culture
- scenario seeds
- survival lore
- superstition and oral-tradition memory

## Repo Rule

Treat tales as source material unless a specific beat, saying, superstition, or event has been explicitly promoted into:

- a canon file
- an imported episode
- a continuity note
