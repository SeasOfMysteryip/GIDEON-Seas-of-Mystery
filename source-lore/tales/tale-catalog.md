# Tale Catalog

This catalog is the quick chooser for the Tales From the Deck source-lore layer.

## Tale Table

| Tale | Type | Origin / Adaptation Importance |
| --- | --- | --- |
| [The Rivalry with Blackbeard](the-rivalry-with-blackbeard.md) | Bloody Gull origin-history / pirate rivalry tale | Major Jeb voice source. Establishes the deck-tale frame, Jeb's preference for wit over brute force, the Bloody Gull's speed and shallows advantage, and Blackbeard as rival pressure rather than simple monster. |
| [The Last Battle of Jebediah Blackwater](the-last-battle-of-jebediah-blackwater.md) | Jeb death-myth / last-stand tale | Important for Jeb's death-shadow, the show's taste for oral-history contradiction, and the recurring idea that the Bloody Gull's legends may be warnings as much as boasts. |
| [The Curse of Black Sand Cove](the-curse-of-black-sand-cove.md) | cursed treasure / survival warning tale | Strong source for cursed-object logic, Jeb's warning monologues, and the crew's inherited rule that treasure is never just treasure. |
| [The Vanishing Merchant of the Triangle](the-vanishing-merchant-of-the-triangle.md) | Triangle disappearance / Bloody Gull mystery tale | Useful for Bermuda/Triangle public myth, real-world disappearance logic, and Jeb's ability to make a route feel like a character. |
| [The Leviathan's Fury](the-leviathans-fury.md) | sea beast / command-under-pressure tale | Feeds Jeb's leadership mode and the show's creature-scale language: awe first, tactics second, spectacle last. |
| [The Kraken's Grasp](the-krakens-grasp.md) | young Jeb / Kraken encounter tale | Good young-Jeb source, useful for showing that his bravado was earned through fear and survival, not born complete. |
| [The Sea Sprites](the-sea-sprites-expanded-tale.md) | sea-sprite / temptation tale | Important for the show's seductive-water logic and for the line between wonder and threat. Also feeds Lysara-adjacent mood without replacing Lysara canon. |
| [The Mermaids of Mistral Bay](the-mermaids-of-mistral-bay-jeb-and-lysara-origin-story.md) | Jeb and Lysara origin-history | Core source-lore for the Jeb/Lysara emotional root. Preserve as adaptation history even where current canon has evolved. |
| [The Specter of the Deep](the-specter-of-the-deep.md) | unknown entity / deep-sea encounter tale | Useful for dread pacing, entity restraint, and the rule that the sea can become wrong before anything appears. |
| [The Leviathan's Wake](the-leviathans-wake.md) | old water / Leviathan awe tale | Good source for old-water language, northern sea dread, and Jeb's awe when the ocean outranks him. |
| [The Ghosts of Nassau](the-ghosts-of-nassau.md) | Nassau pirate history / crew-rebellion tale | Useful for Nassau texture, Jeb's historical placement, and Season Two real-world Nassau scenes where performance and true pirate memory overlap. |
| [Ghost of the Gallows](ghost-of-the-gallows.md) | gallows ghost / condemned pirate tale | Strong source for Jeb's grief-voice and the show's interest in justice, haunting, and rebel memory. |

## Organization Rule

Each tale gets its own file with a use rule, origin/adaptation importance, summary, character DNA, full deck tale, story uses, and canon guardrails. These files preserve the tales as source-history and monologue material rather than disposable folklore.

## Count

The current cleaned tale set contains 12 tales.
