# Source Lore Index

This folder stores origin-layer material that fed the larger `Seas of Mystery` world build before or alongside the current canon system.

It is not hard canon by default.

Use this bucket for:

- adaptation lineage
- source podcasts
- songs and chant material
- tales that later informed characters, ship culture, and scenario design

## Authority Rule

Within `source-lore/`, use authority in this order:

1. converted source files in this folder
2. notes about adaptation usage

If anything in `source-lore/` conflicts with locked files in `canon/` or imported episode material in `episodes/`, the locked canon or episode files win.
