# Source Lore

This folder preserves songs, podcast material, tales, adaptation lineage, and other story-facing folklore sources that can feed scenes without automatically becoming locked canon.

## Use Rule

Source lore is a seed bank. Use it for atmosphere, dialogue texture, public myth, character prompts, local history, sponsor brands, songs, tales, and rumor patterns. Promote material into canon only when it appears in a canon file, episode card, or approved scene record.

## Authority Rule

If anything in `source-lore/` conflicts with locked files in `canon/` or imported episode material in `episodes/`, the locked canon or episode files win.
