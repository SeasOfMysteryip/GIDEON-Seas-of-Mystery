# Podcast Source Lore

These podcast dossiers preserve the usable material from the original Seas of Mystery podcast scripts without storing every full transcript in Gideon's active source path.

## Use Rule

Use podcast files as folklore, public rumor, caller texture, guest formats, recurring in-world sponsor brands, and seed banks for new characters, relics, locations, and episode pressure. Podcast claims are not locked canon unless promoted elsewhere in the source repo.
