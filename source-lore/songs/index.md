# Songs Index

This folder is for songs, chants, dirges, and harmonic fragments that belong to the source-lore layer.

## Special Importance

Songs are a live feeder system for the current IP.

They are especially important for:

- Old Red's line logic and storm-song voice
- relic tuning and stabilization language
- Bloody Gull oral culture
- emotional weather and shipboard memory

## Repo Rule

Do not assume every lyric is literal canon law.

Do preserve:

- recurring refrains
- song function
- who sings what kind of material
- whether a song behaves as warning, stabilization, ritual, memory trigger, or crew culture
