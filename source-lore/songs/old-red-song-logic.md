# Old Red Song Logic

This file captures how the songbook feeds Old Red in the current IP.

## What The Songs Are Doing

The song layer is not just atmosphere. In practice it feeds:

- Old Red's warning language
- Old Red's harmonic-tuning function
- ritual and marrow-deep cadence
- shell / dice / storm synchronization
- shipboard memory and oral tradition

## Old Red Voice Rule

Old Red's lines often feel sourced from:

- broken shanty refrains
- warning chants
- marrow-cycle cautionary songs
- relic-harmonic fragments
- sea-mother / storm-door / don't-look-back language

He does not read as a generic pirate poet.

He reads as someone whose speech has been shaped by:

- old crew song
- omen language
- Navigator warning cycles
- tuning behavior around relic surge

## Recurring Functions

Songs feeding Old Red usually perform one or more of these jobs:

- warn the crew
- stabilize relic overload
- signal a threshold or corridor crossing
- name danger without explaining it clinically
- preserve ship memory across time

## Bridge To Canon

When a song fragment is already embedded in episode scripts or canon files, that use becomes canon-relevant even if the full original lyric remains source-lore.
